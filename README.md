# AWS Static Website Hosting

This project demonstrates how to host a static website on AWS using simple HTML and CSS files.

The website is deployed as a static web application and can be used as a beginner-friendly cloud project to understand AWS static website hosting concepts.

---

## Project Overview

This repository contains a basic static website built using:

- HTML5
- CSS3
- JS

The project focuses on:

- Hosting static content on AWS
- Understanding cloud-based website deployment
- Learning static website architecture
- Practicing beginner-level AWS deployment concepts

---

## Tech Stack

- HTML5
- CSS3
- JS
- AWS Static Website Hosting
